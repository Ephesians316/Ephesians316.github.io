Problems while coding

Before starting the project, my partner and I chose to create a blackjack game in Python. While at first optimistic, I soon realized that the code necessary for blackjack is much more complicated than what my partner and I were capable for. After realizing this I changed the game to become a trivia quiz game. I problem that I faced while writing the code was how to randomize the questions so users will not get the same pattern of questions. After obtaining some assistance from a fellow student I discovered that you can make the questions their own seperate variable and create code that will randomly choose from that list of questions without repeats.
